Detailed Description

Abstract class holding all the variables necessary for both player and enemy types. Also contains basic functions for updating the ship which are universal for both player and enemy types

Member Function Documentation

virtual void PirateWars.Ship.Fire ( )
virtual

Create new cannon balls (cannons * 2) and place them in the CBA. All cannon balls are fired perpindicular to the boat. Cannon number of cannon balls go off the left side, and the same number go off the right. This function first creates the c_direction for a left side cannon ball and then inverts it for the right side Different ships fire different looking cannonBalls, so it is necessary to pass the type of cannonBall into the fire funtion.

Reimplemented in PirateWars.Player_Brig, PirateWars.Player_Frigate, and PirateWars.Player_ManOfWar.
